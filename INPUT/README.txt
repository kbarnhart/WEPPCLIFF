Purpose: Describe contents of the folder 'INPUT'


Content:

AH537_BPT_CP.csv			A sample WEPPCLIFF input file which contains:
						--breakpoint data for a single event
						--cumulative precipitation
						--english units for precipitation
						--filler (fake) daily data
						--depth precision (0.01 or hundredths of inches; common in modern data)

AH537_BPT_IP.csv			A sample WEPPCLIFF input file similar to above but has:
						--incremental precipitation

AH537_1MIN.csv				A sample WEPPCLIFF input file similar to above but has:
						--fixed interval data (1-minute)

AH537_1MIN_UP.csv			A sample WEPPCLIFF input file similar to above but has:
						--unlimited depth precision (not common; just for demonstration)

AH537_5MIN.csv				A sample WEPPCLIFF input file similar to AH537_1MIN.csv but has:
						--fixed interval data (5-minute)

AH537_10MIN.csv				A sample WEPPCLIFF input file similar to above but has:
						--fixed interval data (10-minute)

AH537_15MIN.csv				A sample WEPPCLIFF input file similar to above but has:
						--fixed interval data (15-minute)

AH537_15MIN_UP.csv			A sample WEPPCLIFF input file similar to above but has:
						--unlimited depth precision (not common; just for demonstration)

AH537_15MIN_HT.csv			A sample WEPPCLIFF input file similar to above but has:
						--depth precision (0.1 or tenths of inches; common before 1993)
